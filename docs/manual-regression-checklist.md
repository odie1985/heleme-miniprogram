# 喝了么手动回归清单

适用场景：
- 真机调试前快速检查
- 提审前功能回归
- 分享相关改动后重点复测

## 1. 日历页基础

- 打开首页，确认默认进入日历页
- 今天日期仍然高亮显示
- 有记录的日期仍显示图标标记
- 点击任意日期后，下方显示当天记录
- 点击日期不会跳转到发布页
- 右下角 `+` 按钮可以正常打开快速记录

## 2. 快速记录面板

- 点击 `+` 后，快速记录面板正常弹出
- `喝酒 / 健身` 切换正常
- 喝酒表单包含：`种类`、`场景`、`喝酒的量（备注）`、`状态`
- 健身表单包含：`运动种类`、`运动时长`
- `添加位置` 可正常选点，展示地点名而不是经纬度
- `添加图片` 可正常选图、预览、删除

## 3. 记录保存与展示

- 新建一条带图片和地点的喝酒记录，保存成功
- 回到当天记录列表，可看到：
- 类型标签
- 记录内容
- 喝酒的量
- 地点单独一行显示
- 图片缩略图
- 点击记录条目，会弹出 `修改 / 删除`
- 删除后，列表和日历标记同步更新

## 4. 月份切换与跳转

- 日历左右翻月正常
- 点击年月标题，可按月份快速跳转
- 跳转后对应月份记录仍正常显示

## 5. 我的页与统计

- `我的` 页面可以正常打开
- 统计区正常显示
- 饼图不是空白
- 月份切换或月份导航正常

## 6. 朋友圈分享入口

- 快速记录面板中的 `分享到朋友圈` 按钮可以点击
- 点击后不会直接报错或卡死
- 可以正常进入朋友圈分享流程

## 7. 朋友圈小卡片

- 发一条新的朋友圈分享
- 朋友圈中的小卡片有图片
- 标题文案正常，不是空白

## 8. 朋友圈落地页

- 从朋友圈点开后，落地页能正常打开
- 落地页优先显示分享图片
- 图片上的分享文字信息正常显示
- 不显示 `图片 1 张`
- 喝酒记录会显示 `喝酒的量`
- 页面整体尽量一屏展示，不明显上下滑动
- `进入小程序，立即注册` 按钮文字水平、垂直居中

## 9. 登录与我的页基础

- 未登录状态显示正常
- 点击登录入口逻辑仍可用
- 头像、昵称、签名区域样式正常

## 建议优先回归的 4 个关键点

1. 新建一条带图带地点的喝酒记录并保存
2. 分享到朋友圈，确认小卡片有图
3. 点开落地页，确认图片和喝酒的量正常显示
4. 回到日历页，确认记录仍可修改、删除
